Features
User Authentication: Register, login, and manage your account.
PDF Creation: Generate PDFs from text or form inputs.
Download PDFs: Save your PDFs locally.
Subscription Model:
Free tier: Limited number of PDFs per month.
Premium tier: Unlimited PDF creation, priority features.
Responsive Design: Works on desktop and mobile.
Secure: JWT authentication and protected routes.
Tech Stack
Frontend: React, Vite, TailwindCSS
Backend: Node.js, Express, MongoDB
Authentication: JWT
Payments: Razorpay integration for subscriptions
